## Description

Pool-Pro.uk is a feature rich web application for pool scoring for teams and singles.


## Features
- player names
- team names
- data export
- data import
- custom colours per player 
- persistent data storage
- shot clock
- match clock
- Match Details Table
- Score Tracking & Totals
- clear team data
- restore defaults
- epic auto sizing of long names to fit space
